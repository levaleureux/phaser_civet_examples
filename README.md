# Phaser Examples in Civet

This repository aims to translate all JavaScript examples from [Phaser.io](https://phaser.io/examples) to [Civet](https://civet.dev/), a programming language that compiles to TypeScript/JavaScript, to increase visibility for the Civet language.

## What is Civet?

Civet is a programming language that compiles to TypeScript and JavaScript. It provides a more concise and readable syntax while maintaining compatibility with existing JS/TS ecosystems. Learn more at [civet.dev](https://civet.dev/).

## What is Phaser?

Phaser is a fast, free, and fun open-source HTML5 game framework that offers WebGL and Canvas rendering across desktop and mobile web browsers. It's designed to make the fastest possible 2D games in the browser. Visit [phaser.io](https://phaser.io/) for more information.
